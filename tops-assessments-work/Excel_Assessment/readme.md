# Advanced Excel for Analytics

## Overview
This project was completed as part of the **Advanced Excel for Analytics** assessment. The objective was to apply Excel analytics concepts to solve business problems using PivotTables, formulas, and data analysis techniques.

## Dataset
The assessment uses a sample business transaction dataset for analysis.

## Tasks Completed
- Business KPI Concept Questions
- PivotTable Analysis
- Revenue & Expense Analysis
- Supplier-wise Summary Report
- Basic Business Insights

> **Note:** Power Query tasks were not included in this project, as that topic had not yet been covered during the course.

## Tools Used
- Microsoft Excel
- PivotTables
- Excel Formulas

## Files Included
- `Advaned Excel for Analytics.xlsm` – Excel workbook containing the completed assessment.
