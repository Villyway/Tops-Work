# Python For Data Analytics Assessment

## Overview

This project was completed as part of a Python for Data Analytics assessment. The objective was to perform data cleaning, validation, transformation, and reporting using a retail sales dataset.

## Dataset

Dataset: Sample Sales Data

Source:
https://www.kaggle.com/datasets/kyanyoga/sample-sales-data

## Tasks Performed

### Section A - Concept Application
- String methods and data cleaning concepts
- CSV data type conversion
- Dictionary usage for data mapping
- Loop vs sum() comparison
- Error handling and type checking
- Benefits of modular programming

### Section B - Practical Task
- Loaded the sales dataset
- Cleaned the PRODUCTLINE column
- Converted SALES to numeric format
- Created product-wise sales dictionary
- Classified product lines based on revenue

### Section C - Mini Project
**Title:** Automated Sales Integrity & Reporting Engine

- Data validation
- Missing value analysis
- Data type verification
- Revenue outlier detection
- Automated summary reporting

## Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook
